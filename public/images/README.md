# Project Images

Place your project images in this folder.

## Image Naming Convention

Use the project ID from `constants.tsx` for the filename:

| Project ID | Image Filename |
|------------|----------------|
| latent-spaces | `latent-spaces.jpg` |
| physarum-polycephalum | `physarum.jpg` |
| brutal-glyphs | `brutal-glyphs.jpg` |
| vortex-field | `vortex-field.jpg` |
| noise-orchestra | `noise-orchestra.jpg` |
| recursive-void | `recursive-void.jpg` |

## Supported Formats

- JPG, PNG, WebP, GIF

## Image Size Recommendation

- **Optimal: 1200x800 pixels (3:2 aspect ratio)**
- Minimum: 600x400 pixels
- The image displays at 1/3 width of the project card
- Images are cropped with `object-cover` - important content should be centered

## Adding New Projects

1. Add the project to `constants.tsx`
2. Name your image using the project ID
3. Place the image in this folder
